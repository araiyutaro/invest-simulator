## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Tailwind Class | Usage |
|-------|-------|----------------|-------|
| xs | 4px | `p-1` / `gap-1` | バッジ内パディング、アイコンとラベルの間隔 |
| sm | 8px | `p-2` / `gap-2` | カード内テキスト間隔、インラインボタン padding |
| md | 16px | `p-4` / `gap-4` | カードデフォルトパディング、フォーム要素間隔 |
| lg | 24px | `p-6` / `gap-6` | セクション内パディング、テーブル行間 |
| xl | 32px | `p-8` / `gap-8` | ページ水平パディング（現行 `p-8` を踏襲） |
| 2xl | 48px | `py-12` / `gap-12` | セクション間の垂直マージン |
| 3xl | 64px | `py-16` | ページトップ余白 |

Exceptions:
- ポートフォリオ推移チャートの高さ: 400px（固定値。8pxグリッドの例外として明示）
- タッチターゲット最小高さ: 44px（「もっと見る」ボタン、Sign out ボタン）

## Typography

| Role | Size | Weight | Line Height | Font | Usage |
|------|------|--------|-------------|------|-------|
| Body | 14px | 400 (regular) | 1.6 | Geist Sans | reasoning テキスト、market_assessment 本文 |
| Label | 12px | 400 (regular) | 1.4 | Geist Sans | バッジラベル（high/medium/low）、テーブルヘッダー、カード指標名。色差（text-slate-400）で視覚的区別を行う |
| Heading | 20px | 600 (semibold) | 1.2 | Geist Sans | セクション見出し（「ポートフォリオ推移」等） |
| Display | 28px | 600 (semibold) | 1.1 | Geist Mono | パフォーマンス指標カードの数値（累計リターン%等） |

**フォント補足:**
- 数値（パフォーマンス指標、価格、損益）は全て Geist Mono で統一。桁数変動時の横幅ズレを防ぐ
- Tailwind クラス: `text-sm` (14px), `text-xs` (12px), `text-xl` (20px), `text-3xl` (28px)
- 重みクラス: `font-normal` (400), `font-semibold` (600)
- **500 (medium) は使用しない。** Label の視覚的区別は `text-xs` + `text-slate-400` の色差で行う

## Layout Contract

### ページ構造（CONTEXT.md D-01, D-02 確定）

```
<header>                   — アプリ名 + Sign out ボタン (h-14, bg-slate-900, border-b border-slate-800)
<main class="max-w-7xl mx-auto px-8 py-12">
  Section 1: PerformanceGrid      — 3×2 指標カードグリッド
  Section 2: PortfolioChart       — 推移ラインチャート (full-width)
  Section 3: PositionsRow         — PositionsTable (左60%) + AllocationChart (右40%)
  Section 4: TradeTimeline        — 日付単位タイムライン（無限スクロール風ページネーション）
</main>
```

- 最大幅: `max-w-7xl`（1280px）、中央寄せ
- 水平パディング: 32px (`px-8`)
- セクション間垂直マージン: 48px (`space-y-12`)
- レスポンシブブレイクポイント: `lg:` (1024px) でポジションテーブル+パイチャートを横並び。それ未満は縦積み

**Primary focal point:** Section 1 PerformanceGrid の Display 数値（28px Geist Mono）が主視覚アンカー。ユーザーが最初に目を向けるべき情報を最上部・最大フォントサイズで示す。

### PerformanceGrid（CONTEXT.md D-17 確定）

**Type:** Server Component  
**File:** `app/dashboard/components/PerformanceGrid.tsx`

3列×2行グリッド。`grid grid-cols-3 gap-4`

| Position | Metric | Unit | Color Rule |
|----------|--------|------|------------|
| [0,0] | 累計リターン | % | ±で green-400 / red-400 |
| [0,1] | vs SPY差分 | % | ±で green-400 / red-400 |
| [0,2] | シャープレシオ | 小数点2桁 | ±で green-400 / red-400 |
| [1,0] | 最大ドローダウン | % (負値) | 常に red-400 |
| [1,1] | 勝率 | % | >=50%: green-400 / <50%: red-400 |
| [1,2] | 取引数 | 整数 | slate-300 (ニュートラル) |

**カード構造:**
```
<div class="bg-slate-800 rounded-lg p-6">
  <p class="text-xs text-slate-400 mb-2">{指標名}</p>
  <p class="text-3xl font-semibold font-mono {color}">{値}</p>
</div>
```

**Empty state:** データ未蓄積時は全カードに `—` を slate-400 で表示

### PortfolioChart（CONTEXT.md D-05, D-06 確定）

**Type:** Client Component (`"use client"`)  
**File:** `app/dashboard/components/PortfolioChart.tsx`

- ライブラリ: `lightweight-charts-react-wrapper` v2.1.1

**Empty state:** チャートエリアに `"運用開始後にデータが表示されます"` を中央表示（slate-400）

### TradeTimeline（CONTEXT.md D-11〜D-15 確定）

**Type:** Client Component（アコーディオン状態管理のため）  
**File:** `app/dashboard/components/TradeTimeline.tsx`

**日付グループ構造:**
```
<section>
  <h3>{YYYY年MM月DD日（曜日）}</h3>         — text-sm font-normal text-slate-400
  <p class="market-assessment">{テキスト}</p> — text-sm text-slate-300 bg-slate-800 rounded p-4 mb-4
  [取引なし の場合]
    <p class="text-sm text-slate-500 italic">この日の取引なし</p>
  [BUY/SELL が存在する場合]
    <TradeCard /> × n
</section>
```

**TradeCard 構造:**
```
<div class="bg-slate-800 rounded-lg p-4 mb-3">
  <div class="flex justify-between items-center">
    <span class="ticker font-mono text-slate-100">{TICKER}</span>
    <span class="action-badge">{BUY|SELL}</span>      — BUY: bg-blue-900/40 text-blue-400 / SELL: bg-slate-700 text-slate-300
    <span class="confidence-badge">{high|medium|low}</span>
    <span class="font-mono text-slate-300">{quantity}株 @ ¥{price}</span>
  </div>
  <details open>                                        — デフォルト展開（CONTEXT.md D-13）
    <summary class="text-xs text-slate-400 cursor-pointer mt-2">判断理由</summary>
    <p class="text-sm text-slate-300 mt-2 leading-relaxed">{reasoning}</p>
  </details>
</div>
```

**ページネーション（CONTEXT.md D-15 確定）:**
- 初期表示: 直近20日分
- 追加読み込み: 「さらに読み込む」ボタンで20日分追加（合計上限なし）
- ボタン: `bg-slate-800 hover:bg-slate-700 text-slate-300 rounded-lg px-6 py-3 text-sm`

**全HOLDの日の表示:** market_assessment + `"この日の取引なし"` テキスト（CONTEXT.md D-12 確定）

## Accessibility Notes

- `<details open>` で reasoning をデフォルト展開（ネイティブHTML、JS不要）
- `<summary>` はキーボードフォーカス可能
- 確信度バッジは色のみでなくテキスト（HIGH/MED/LOW）も併記（色覚バリアフリー対応）
- BUY/SELLバッジも同様にテキスト明示
- チャートは Canvas 描画のため `aria-label` を `<div>` ラッパーに付与: `aria-label="ポートフォリオ推移チャート"`
