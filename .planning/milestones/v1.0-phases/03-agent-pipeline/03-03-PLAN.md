**関数シグネチャ:**
```typescript
export type ExecuteParams = {
  readonly decisions: readonly GeminiDecisionItem[]
  readonly portfolioId: string
  readonly decisionId: string
  readonly closePrices: ReadonlyMap<string, number>  // symbol → close price
  readonly fxRateUsdJpy: number | null
  readonly currentCashJpy: number
  readonly currentPositions: ReadonlyMap<string, { quantity: number; avgCost: number; currency: 'USD' | 'JPY' }>
}

export async function executeDecisions(params: ExecuteParams): Promise<ExecutionResult>
```

**実装の詳細:**

1. HOLDを除外: `decisions.filter(d => d.action !== 'HOLD')`

2. 各BUY/SELL判断を順番に処理（`let runningCash = currentCashJpy` で可変追跡）:

   **BUYの場合（D-08, D-09, D-10）:**
   - `const closePrice = closePrices.get(d.ticker)` — nullなら skip (reason: 'no_close_price')
   - `const ticker = findTicker(d.ticker)` — currency取得
   - USD銘柄: `costJpy = closePrice * d.quantity * (fxRateUsdJpy ?? 0)`、fxRateUsdJpy nullなら skip
   - JPY銘柄: `costJpy = closePrice * d.quantity`
   - `runningCash < costJpy` なら skip (reason: 'insufficient_cash')
   - `runningCash -= costJpy`
   - avgCost計算: 既存ポジションがあれば加重平均、なければ約定価格
     ```
     newAvgCost = (existQty * existAvgCost + buyQty * closePrice) / (existQty + buyQty)
     ```
   - DB操作:
     - `trades` にINSERT: `{ portfolioId, decisionId, symbol, action: 'BUY', quantity, executedPrice: closePrice.toString(), commission: '0', currency, fxRateToJpy }`
     - `positions` にUPSERT: `onConflictDoUpdate` で `quantity = existing + buyQty`, `avgCost = newAvgCost`, `updatedAt = now`

   **SELLの場合（D-09, D-11）:**
   - 保有数量チェック: `existingPosition.quantity < d.quantity` なら skip (reason: 'insufficient_shares')
   - USD銘柄: `proceedsJpy = closePrice * d.quantity * (fxRateUsdJpy ?? 0)`
   - JPY銘柄: `proceedsJpy = closePrice * d.quantity`
   - `runningCash += proceedsJpy`
   - DB操作:
     - `trades` にINSERT: `{ action: 'SELL', ... }`
     - `positions` をUPDATE: `quantity = existing - sellQty`, `updatedAt = now`
     - quantity=0になっても行を削除しない（D-11）

3. `portfolios` の `cash` カラムを `runningCash` で UPDATE

4. 戻り値: `{ trades, skipped, newCashJpy: runningCash }`
