**exports:**

```typescript
export type GeminiCallResult =
  | { ok: true; response: GeminiResponse; filteredDecisions: GeminiDecisionItem[]; usage: { promptTokens: number; candidateTokens: number; totalTokens: number }; costUsd: number; rawText: string }
  | { ok: false; error: string; rawText?: string }
```

**`callGemini(systemPrompt: string, userPrompt: string): Promise<GeminiCallResult>`**

1. Geminiモデル設定（D-05, D-07）:
   - `genAI.getGenerativeModel()` に以下を渡す:
     - `model: GEMINI_MODEL`
     - `generationConfig.responseMimeType: 'application/json'`
     - `generationConfig.responseSchema`: SchemaTypeを使ったD-07のスキーマ定義
       ```typescript
       import { SchemaType } from '@google/generative-ai'
       const responseSchema = {
         type: SchemaType.OBJECT,
         properties: {
           market_assessment: { type: SchemaType.STRING, description: '全体の市場環境分析（日本語）' },
           decisions: {
             type: SchemaType.ARRAY,
             items: {
               type: SchemaType.OBJECT,
               properties: {
                 ticker: { type: SchemaType.STRING },
                 action: { type: SchemaType.STRING, description: 'One of: BUY, SELL, HOLD' },
                 quantity: { type: SchemaType.INTEGER, description: '売買株数。HOLDの場合は0' },
                 confidence: { type: SchemaType.STRING, description: 'One of: high, medium, low' },
                 reasoning: { type: SchemaType.STRING, description: 'この判断の理由（日本語）' },
               },
               required: ['ticker', 'action', 'quantity', 'confidence', 'reasoning'],
             },
           },
         },
         required: ['market_assessment', 'decisions'],
       }
       ```
     - `generationConfig.temperature: 0.3`
     - `systemInstruction: systemPrompt`

2. API呼び出しとリトライ（D-14）:
   - `model.generateContent(userPrompt)` を実行
   - 例外発生時: 30秒待機（`await new Promise(r => setTimeout(r, 30_000))`）して1回リトライ
   - 2回目も失敗: `{ ok: false, error: エラーメッセージ }` を返す

3. レスポンス処理:
   - `result.response.text()` でrawテキスト取得
   - `JSON.parse(rawText)` でパース
   - `GeminiResponseSchema.parse(parsed)` でzodバリデーション
   - バリデーション失敗時: `{ ok: false, error: 'validation_failed: ...' }` を返す

4. フィルタリング（D-15）:
   - `findTicker(d.ticker)` でホワイトリスト外を除外
   - `d.action === 'HOLD'` はフィルタ済み配列に含めるが、executorで使うのはBUY/SELLのみ
   - スキップされた判断の理由をログ出力

5. コスト推定（AGENT-07）:
   - `result.response.usageMetadata` から `promptTokenCount`, `candidatesTokenCount`, `totalTokenCount` を取得
   - `estimateTokenCostUsd()` でUSDコスト計算
