実装する関数:

a. `parseTimelineFromDecision(transcript: unknown): { marketAssessment: string; trades: TimelineTrade[] }`
   - `transcript` を `as { market_assessment?: string; decisions?: unknown[] }` でキャスト
   - `decisions` が配列でなければ `{ marketAssessment: '', trades: [] }` を返す
   - 配列をフィルタ: `action === 'BUY' || action === 'SELL'` のみ
   - 各エントリを TimelineTrade に変換

b. `getPortfolioId(): Promise<string>`
   - `db.select({ id: portfolios.id }).from(portfolios).limit(1)` でID取得
   - 見つからない場合は Error を throw

c. `getChartData(portfolioId: string): Promise<{ portfolio: {date: string; value: number}[]; spy: {date: string; value: number}[]; topix: {date: string; value: number}[] }>`
   - portfolioSnapshots から snapshotDate 昇順で totalValueJpy 取得
   - priceSnapshots から SPY の close を priceDate 昇順で取得（marketClosed=false のみ）
   - priceSnapshots から 1306.T の close を priceDate 昇順で取得（marketClosed=false のみ）
   - 各系列の numeric 文字列を Number() 変換

d. `getPositionsWithPrices(portfolioId: string): Promise<{ positions: PositionWithPrice[]; allocations: AllocationSlice[]; cash: number }>`
   - positions テーブルで portfolioId フィルタ、quantity > 0 のみ
   - 各 symbol について priceSnapshots から最新の close を取得（marketClosed=false、priceDate DESC LIMIT 1）
   - portfolios テーブルから cash を取得
   - unrealizedPnl = (currentPrice - avgCost) * quantity。USD銘柄の場合は priceSnapshots の fxRateToJpy で JPY 換算
   - allocation = 各銘柄の時価評価額JPY / (全銘柄時価評価額JPY + 現金)  * 100
   - AllocationSlice 配列を生成（現金は最後のエントリ、name='CASH'）

e. `getTimelineData(portfolioId: string, limit: number = 20, offset: number = 0): Promise<TimelineDay[]>`
   - decisions テーブルで portfolioId フィルタ、runDate DESC、LIMIT + OFFSET
   - 各 decision に対して trades テーブルを JOIN して実際に約定した取引の executedPrice/quantity を取得
   - `parseTimelineFromDecision` で transcript を解析
   - transcript の decisions[] と trades テーブルの結果をマージ: transcript から confidence/reasoning、trades から executedPrice/quantity

f. `getPerformanceData(portfolioId: string): Promise<{ snapshots: ...; spySnapshots: ...; trades: ...; positionAvgCosts: ... }>`
   - calculateMetrics に渡すデータを一括取得
   - portfolioSnapshots、SPY priceSnapshots、trades、positions（avgCost マップ用）
